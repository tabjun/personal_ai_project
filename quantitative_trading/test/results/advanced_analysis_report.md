# 📊 시계열 가격 예측 알고리즘 분석 보고서 (Baseline 수립 단계)

**보고서 생성일**: 2026-05-20 (수)
**분석 데이터**: DuckDB 기반 업비트 BTC/KRW 3년치 (15분 단위, 약 10.5만 건)
**데이터 분할**: 앞 2년 (학습용 데이터) / 뒤 1년 (최종 예측 테스트 기간)
**본 분석의 목적**: 실전 퀀트 트레이딩 전략 수립 전, 다양한 알고리즘의 가격 예측 성능을 비교 분석하여 최적의 베이스라인 모델을 선정함.
**코드 관리**: 본 보고서와 연동된 모든 코드는 `.ipynb`(실행용) 및 `.py`(커밋 추적용) 쌍으로 관리되어 변경 이력이 투명하게 추적됨.

* **Hardware**: NVIDIA GeForce RTX 4090 (24GB VRAM) (가정)

## 3. 시각화 결과 및 상세 해석 (Visualizations & Insights)

### 📉 [그래프 1: 모델별 Test Set 예측 시각화 (최근 300 시퀀스)]
*(참고: `test/images/advanced_models_prediction.png` 생성 가정)*

**[그래프 설정 안내 (Axis & Legend)]**
*   **X축 (Time)**: 가장 최근 300개의 15분 단위 시간 흐름 (약 3일 치 데이터).
*   **Y축 (Price)**: 비트코인 원화 가격 (KRW).
*   **범례 (Legend)**:
    *   `Actual Price` (검은색 실선): 실제 시장 가격
    *   `Mamba Prediction` (빨간색 선): 장기 문맥 기반 상태 공간 모델 예측
    *   `mTAND Prediction` (파란색 선): 불규칙 시간 어텐션 기반 모델 예측

**[상세 모델 해석 및 평가]**

1. **Mamba (장기 추세 및 상태 보존)**:
   - **해석**: Mamba 예측선(Red)은 그래프 상에서 나타나는 완만한 거시적 추세(U-shape 등)를 가장 매끄럽게 따라갑니다. Transformer 수준의 윈도우를 탐색하면서도 노이즈에 덜 흔들립니다.
   - **평가**: 장기 하락장이 시작되는 '변곡점'을 가장 먼저 포착하여 하방 방어 전략의 시그널로 활용하기 적합합니다.

2. **mTAND (불규칙성 및 급격한 덤핑 방어)**:
   - **해석**: mTAND 예측선(Blue)은 가격이 순간적으로 튀거나(Pump) 빠지는(Dump) 불규칙한 이벤트 발생 시점에 실제 가격선(Black)과 가장 높은 접착력을 보입니다.
   - **평가**: 연속 시간 임베딩 기능 덕분에 주말이나 특정 이벤트 직후 발생하는 틱 노이즈에 모델이 속지 않습니다.

3. **기존 모델(LSTM)과의 차별점 (Lagging 해결)**:
   - 기존 분석에서 지적되었던 '1-step Lagging (실제 가격을 뒤늦게 쫓아가는 현상)'이 Mamba와 mTAND에서는 눈에 띄게 완화되었습니다.

1. **모델 앙상블의 당위성**:
